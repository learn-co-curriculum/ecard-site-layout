# ECard Site Layout

Create the Ecard site using the screenshot as a guide.

## Tasks

1. Fork this repo
2. Clone your fork.
3. cd to your local cloned copy.
4. Use the screenshot.png image as a guide to write the HTML in index.html and CSS in style.css to match the image.
